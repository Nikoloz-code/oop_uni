# კლასის სტატიკური მთვლელი. მასზე წვდომა, ინიციალიზება, თვალსაჩინო მაგალითები.

სტატიკური წევრები კლასში იმის ნაცვლად რომ ყველა ობიეტს თავ-თავისი ჰქონდეს,
ყველას ერთი აქ, ანუ თუ ერთი ობიექტისთვის რამე მნიშვნელობა აქვს ამ წევრს, ყველა
სხვა ობიექტისთვისაც იგივე მნიშვნელობა ექნება. ეს ფუნქციონალი შეგვიძლია 
გამოვიყენოთ არსებული ობიექტების დასათვლელად.


### სტატიკურ წევრზე წვდომა და ინიციალიზაცია

სტატკიურ წევრზე წვდომა შეგვიძლია გვქონდეს ::[^1] ოპერატორით შემდეგი ფორმით:

```cpp
ცვლადის_ტიპი კლასის_სახელი::ცვლადის სახელი = მნიშვნელობა;
```

### მაგალითი
```cpp
#include <iostream>

class Shape {
private:
    static int object_count; 
public:
    Shape() {
        object_count++; 
    }
    static int get_object_count() {
        return object_count; 
    }
};

int Shape::object_count = 0;

int main() {
    Shape obj1;
    Shape obj2;
    Shape obj3;

    std::cout << "object count: " << Shape::get_object_count() << '\n';
}
```


[^1]: Scope Resolution operator
